# StageCraft Frontend

## Available Scripts

In the project directory, you can run:

### `npm start`

Starts the Vite development server.

### `npm run dev`

Starts the Vite development server.

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

Vite will choose the next free port if 3000 is already in use.

### `npm test`

Runs the Vitest test suite once.

### `npm run build`

Builds the app for production to the `dist` folder.

### `npm run preview`

Serves the production build locally for verification.

## Notes

This frontend was migrated from Create React App to Vite to remove outdated vulnerable build-time dependencies while keeping the existing React application code intact.
